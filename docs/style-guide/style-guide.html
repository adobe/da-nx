EW Style Guide

A living reference of reusable CSS patterns (nx2/styles/form.css, nx2/styles/buttons.css) and shared UI components (nx2/blocks/shared/). Each example is rendered live from the real stylesheets/components, so it can't drift out of sync — copy the snippet shown under each one. Full API docs for each component live under docs/ and are linked inline.

Buttons Form Shared Blocks

Buttons

Source: nx2/styles/buttons.css

Buttons

<button class="nx-btn-primary">Primary</button>
<button class="nx-btn-secondary">Secondary</button>

Action button (small)

<button type="button" class="nx-action-btn">Label</button>

<!-- with icon -->
<button type="button" class="nx-action-btn">
  <svg viewBox="0 0 20 20" aria-hidden="true">
    <use href="path_to_icon"></use>
  </svg>
  <span>Label</span>
</button>

Action button quiet (small)

<button type="button" class="nx-action-btn-quiet">Label</button>

<!-- with icon -->
<button type="button" class="nx-action-btn-quiet">
  <svg viewBox="0 0 20 20" aria-hidden="true">
    <use href="path_to_icon"></use>
  </svg>
  <span>Label</span>
</button>

Action button icon only (small + quiet)

<button type="button" class="nx-action-btn-icon" aria-label="More actions">
  <svg viewBox="0 0 20 20" aria-hidden="true">
    <use href="path_to_icon"></use>
  </svg>
</button>

<!-- toggled/selected state -->
<button type="button" class="nx-action-btn-icon is-active" aria-label="More actions">...</button>

Form

Source: nx2/styles/form.css

Text input

<input class="nx-input" type="text" placeholder="Placeholder text"/>

Form field with label + error

<label class="nx-form-field nx-field-error">
  Email
  <input class="nx-input" type="email" value="not-an-email"/>
  <span class="nx-input-error-msg">Enter a valid email address.</span>
</label>

Checkbox

<label class="nx-checkbox"><input type="checkbox"/> Unchecked</label>
<label class="nx-checkbox"><input type="checkbox" checked/> Checked</label>
<label class="nx-checkbox indeterminate"><input type="checkbox"/> Indeterminate</label>
<label class="nx-checkbox loading"><input type="checkbox" disabled/> Loading</label>

<!-- --nx-checkbox-size overrides the checkbox's width/height (default 14px) -->
<label class="nx-checkbox" style="--nx-checkbox-size: 20px;"><input type="checkbox" checked/> Large</label>

Loading spinner

<span class="nx-loading-spinner" role="status" aria-label="Loading"></span>

Shared Blocks

Source: nx2/blocks/shared/ — see linked docs for full props/events/slots.

nx-menu docs →

<nx-menu id="my-menu">
  <button slot="trigger">Options</button>
</nx-menu>

menu.items = [
  { id: 'edit', label: 'Edit' },
  { id: 'delete', label: 'Delete' },
];
menu.addEventListener('select', (e) => console.log(e.detail.id));

nx-dialog docs →

<nx-dialog title="Delete item" @close=${onClose}>
  <p>This action cannot be undone.</p>
  <button slot="actions" class="nx-btn-secondary">Cancel</button>
  <button slot="actions" class="nx-btn-primary">Delete</button>
</nx-dialog>

nx-popover docs →

const popover = document.createElement('nx-popover');
container.append(popover);
popover.innerHTML = '<p>Popover content</p>';
button.addEventListener('click', () => popover.show({ anchor: button }));
